## JUST DO IT 
> 모르거나 헷갈리는 것을 정리하며 나아가기 

- 커스텀 랜더패스는 랜더 단계 어디에 들어가는가?

```
불투명 오브젝트 렌더링 (큐브, 벽, 바닥...)
    ↓
투명 오브젝트 렌더링 (유리, 파티클...)
    ↓
💥 Comic Book 패스 실행! 💥
    - 화면 전체를 가져와서
    - 만화 효과 적용 (포스터화, 윤곽선)
    - 새 텍스처로 교체
    ↓
URP 포스트 프로세싱 (Bloom, Tone Mapping...)
    ↓
최종 화면
```

- 오브젝트 관련된 랜더링을 마치고 스크린 랜더 부분에 들어간다고 생각하면 됨

- 랜더 피처를 다중으로 껴넣어도 괜찮나?
	- 하나로 통합해서 사용하는 것이 좋은 것 같음

- 포스트 프로세싱과의 차이?
	- 기본 제공 효과만 사용 가능
	- 순서 제어가 어려움


- 언제 쓰면 좋을까?

✅ URP에 없는 특수 효과
   - 만화 스타일, 포스터화
   - 열화상, 야간투시경
   - 글리치, 왜곡
   
✅ 정밀한 제어 필요
   - 렌더링 순서 중요
   - 특정 오브젝트만 처리
   
✅ 고급 기법
   - 듀얼 카메라, 포탈
   - 커스텀 라이팅
   
✅ 최적화 직접 관리
   - 모바일 VR 최적화
